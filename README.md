# 🤖 AI RAG Research Assistant

A Retrieval-Augmented Generation (RAG) application that answers user questions using custom documents and Large Language Models (LLMs).

---

## 🚀 Features

- Document ingestion (PDF, TXT)
- Semantic search using vector embeddings
- Retrieval-Augmented Generation (RAG)
- LangChain integration
- OpenAI / Llama model support
- Context-aware question answering

---

## 🛠️ Tech Stack

- Python
- LangChain
- FAISS
- OpenAI API
- Hugging Face
- Pandas
- NumPy
